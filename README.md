# kmzTOsctConverter
Flask App to converter .kmz files to Euroscope Sector Files (.sct)

### Usage:
  Create your .kmz file by google earth with the next tags:
  
    Taxiways = "Taxyways_..."
    Background = "..._Background"
    Runway = "..._Runway"
    Marking = "Marking..."
    ApronMark = "ApronMark..."
    Taxiway_ = "Taxiway_..."
    stop  = ".._stop"
    Stand_ = "Stand_..."
    Relva : = "Relva..."
    Building = "Building_..."
    
  Open your .kmz file with kmzTOsctConverter and get sct output
